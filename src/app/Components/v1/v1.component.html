Ivy

At this project I use Ivy. What is Ivy? Ivy is a very important stepping stone in the Angular history. It changes how the framework internally works, without changing how we write Angular applications. If the parallel makes some sense to you, it’s very similar to React and the “Fiber rewrite”. React Fiber was a complete rewrite of React internals, and notably offered a more incremental rendering. The rewrite lasted over a year, and opened the door to new features (for example the famous Hooks that were released in React 16.8 and that rely on Fiber).

Store

We use store. This store is motivated by Todd Motto, Google developer who work on creating Angular freamwork and this is made step by step without NgRx plugin. Usually in Angular apps we use NgRx (like in React Redux or Mobx) for state control and it's based on Flux pattern.
